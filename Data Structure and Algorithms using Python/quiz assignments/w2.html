x = [1,"abcd",2,"efgh",[3,4]]  # Statement 1
y = x[0:6]                     # Statement 2
z = x                          # Statement 3
w = y                          # Statement 4
x[1] = x[1][0:3] + 'd'         # Statement 5
y[2] = 4                       # Statement 6
z[1][1:3] = 'yzw'              # Statement 7
z[0] = 0                       # Statement 8
w[4][0] = 1000                 # Statement 9
a = (x[4][1] == 4)             # Statement 10

Yes, the answer is correct.
Score: 2.5

Consider the following lines of Python code. 
x = [423,'b',37,'f']
u = x[1:]
y = u
w = x
u = u[0:]
u[1] = 53
x[2] = 47
After these execute, which of the following is correct?

Yes, the answer is correct.
Score: 2.5

Feedback:

x and w are aliased. u and y are not, after the assignment u = u[0:]. So the update to x[2] affects w[2] while the update to u[1] leaves y[1] unchanged.

What is the value of list1 after the following lines are executed? 
def mystery(l):
  l = l[0:5]
  return()

list1 = [44,71,12,8,23,17,16]
mystery(list1)

No, the answer is incorrect.
Score: 0

Feedback:

Since l is reassigned a value within the function, it refers to a new value and the update does not affect the value of list1 outside the function. [44, 71, 12, 8, 23, 17, 16]
